### 문제 설명

---

함수 solution은 정수 x와 자연수 n을 입력 받아, x부터 시작해 x씩 증가하는 숫자를 n개 지니는 리스트를 리턴해야 합니다. 다음 제한 조건을 보고, 조건을 만족하는 함수, solution을 완성해주세요.

---

### 제한 조건

- x는 -10000000 이상, 10000000 이하인 정수입니다.
- n은 1000 이하인 자연수입니다.

### 해결 방법

---

- 입출력 예를 통한 규칙이 `x*i`
- `for` 구문을 돌려서 answer 에 append 해줌

---

### 코드

---

```python
def solution(x, n):
    answer = []
    for i in range(n):
        answer.append(x*(i+1))
    return answer
```
